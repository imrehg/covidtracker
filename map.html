Hotspot Map

This map shows the weekly prevalence, per 100,000 population, of COVID-19 across the UK for the last seven days that reliable data is available. You can change the metric shown using the switcher on the right.

Click on an area on the map to see the current numbers and a history of the last 45 days of daily cases.

Sources

Data is reported by date of sample, which means that some cases may be missing for the most recent few days. When calculating the weekly prevalence, the most recent 5 days of data are ignored unless they would cause the prevalence number to increase. This allows spikes in cases to be detected early.
